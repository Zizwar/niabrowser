# Screenshots Guide - NIABrowser

## Required Sizes

### Google Play
- Phone: min 320px, max 3840px, 16:9 or 9:16 ratio
- Minimum 4 screenshots, maximum 8
- Recommended: 1080 x 1920 (portrait)

### App Store
- iPhone 6.7" display: 1290 x 2796
- iPhone 6.5" display: 1242 x 2688
- iPad 12.9": 2048 x 2732 (if supporting iPad)
- Minimum 3 screenshots

## Screenshot Plan (8 screens)

### 1. New Tab Dashboard
**What to show:** Open the app — the new tab page with NIA logo, search bar, tool buttons (AI Chat, Scripts, DevTools, API Client, Settings, About), and quick links
**Caption EN:** "Your developer command center"
**Caption AR:** "مركز قيادة المطور"
**Tips:** Use dark mode. Make sure all 6 tool buttons are visible.

### 2. AI Chat in Action
**What to show:** AI Chat open with a real conversation. User asks about the current page, AI responds with analysis. Show attachment chips enabled (network, console).
**Caption EN:** "Chat with AI about any page"
**Caption AR:** "تحدث مع الذكاء الاصطناعي عن اي صفحة"
**Tips:** Have 2-3 messages visible. Show the model selector at the top.

### 3. Network Monitor
**What to show:** DevTools panel open on Network tab. Show a list of captured requests with method badges (GET, POST), status codes (200, 304), URLs, and timing.
**Caption EN:** "Monitor every network request"
**Caption AR:** "راقب كل طلبات الشبكة"
**Tips:** Visit a real site first to capture diverse requests.

### 4. Script Manager
**What to show:** Script list with 2-3 scripts. Show toggle switches, play/edit/delete buttons. One script enabled, one disabled.
**Caption EN:** "Automate with smart scripts"
**Caption AR:** "اتمتة مع سكريبتات ذكية"
**Tips:** Create sample scripts with clear names like "Dark Mode Theme", "Remove Ads".

### 5. AI Script Generator
**What to show:** The AI Generator screen. Task description field with text like "Change all images to grayscale". Show generated code in the editor below.
**Caption EN:** "Describe it — AI writes the code"
**Caption AR:** "صف ما تريد - الذكاء يكتب الكود"
**Tips:** Use a clear, simple task description.

### 6. API Client
**What to show:** CRUD modal with a POST request. Show URL, method selector, headers tab, body with JSON, and a response section.
**Caption EN:** "Test APIs on the go"
**Caption AR:** "اختبر الواجهات البرمجية"
**Tips:** Use a public API like jsonplaceholder.typicode.com.

### 7. Settings - AI Providers
**What to show:** Settings screen on AI tab. Show provider tabs (OpenRouter, Google Gemini with FREE badge, OpenAI), model list below with costs.
**Caption EN:** "100+ AI models, your choice"
**Caption AR:** "اكثر من 100 نموذج ذكاء اصطناعي"
**Tips:** Show the model list with different providers' models visible.

### 8. Console & Multi-Tab
**What to show:** Split view — console output visible in DevTools panel (with colored log, warn, error messages) and multiple tabs visible at top.
**Caption EN:** "Full DevTools suite on mobile"
**Caption AR:** "ادوات مطور كاملة على الجوال"
**Tips:** Visit a JS-heavy site to get colorful console output.

## Tips for Taking Screenshots

1. **Use a clean device** or emulator with no notifications in status bar
2. **Dark mode ON** — looks more professional and matches app identity
3. **Use real content** — visit real websites, don't use placeholder data
4. **Consistent style** — same theme, same font size across all screenshots
5. **Add frames** (optional) — Use a tool like screenshots.pro or mockuphone.com to add device frames
6. **Add captions** — Use Figma, Canva, or similar to add text captions below/above each screenshot
